# Realtime Mouse Tracker App

Учебное приложение на **React + TypeScript** и **WebSocket**, которое показывает обмен состоянием между клиентами в реальном времени.

## Структура проекта

```
project-root/
├── client/   # фронтенд на React
├── server/   # бэкенд на Node.js (ws)
│   └── index.js
```

## Запуск

### 1. Сервер

В папке `server` уже есть готовый файл `index.js`.

```bash
cd server
npm install
node index.js
```

Сервер стартует на `ws://127.0.0.1:8000`.

### 2. Клиент

```bash
cd client
npm install
npm run dev
```

После запуска открой [http://localhost:5173](http://localhost:5173) (или порт, который выдаст Vite).

## Как работает

- При входе пользователь вводит имя.
- Клиент подключается к WebSocket‑серверу (`server/index.js`).
- При движении мыши координаты (`x`, `y`) отправляются на сервер.
- Сервер рассылает обновлённые состояния всем подключённым клиентам.
- На фронтенде отображается список пользователей и их позиции.

## Технологии

- **React + TypeScript**
- **react-use-websocket**
- **Node.js + ws**
- **Throttle** для оптимизации отправки событий

---

Приложение можно использовать как основу для:

- простых многопользовательских игр,
- совместных редакторов,
- визуализации активности пользователей в реальном времени.
